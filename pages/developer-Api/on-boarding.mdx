### On-boarding

This API provides endpoints for accepting payments in your applications. Below, you'll find detailed information on how to integrate and use the API.

### Authentication

You need an API key. You can obtain your API key by signing up for an account on our platform. Use this API key in the Authorization header of your HTTP requests.

<Callout type="warning" emoji="⚠️">
 Make sure to never post your API publicly
</Callout>

<Onboarding/>



```json
{
  "request": {
    "method": "POST",
    "url": "https://api.example.com/data",
    "headers": {
      "Authorization": "Bearer YOUR_API_KEY"
    }
  }
}
```
